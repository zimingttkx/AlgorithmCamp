# AlgorithmCamp LeetCode 刷题系统增强

## 目标
深度优化 LeetCode 刷题体验，添加智能推荐、进度追踪、数据分析等功能。

## 任务

- [x] 1. LeetCode 用户绑定
  - 添加 LeetCode 用户名输入
  - 验证用户名有效性
  - 展示 LeetCode 官方头像和基本信息

- [x] 2. 解题记录同步
  - 自动同步 LeetCode 已解题
  - 手动标记本地已解题
  - 同步状态展示（已同步/待同步）

- [x] 3. 智能题目推荐
  - 基于章节的推荐算法
  - 基于难度的推荐算法
  - 基于遗忘曲线的复习提醒
  - 推荐算法说明页面

- [x] 4. 进度可视化升级
  - 添加 3D 进度地球/城市视图
  - 添加刷题路线图可视化
  - 添加能力雷达图
  - 添加解题时长统计

- [x] 5. 刷题统计面板
  - 总解题数趋势图
  - 各类目占比饼图
  - 难度分布柱状图
  - 周/月刷题热力图

- [x] 6. 复习提醒系统
  - 基于 SM-2 算法安排复习
  - 标记需要复习的题目
  - 邮件/站内通知提醒

- [x] 7. 题目详情页
  - 题目链接到 LeetCode
  - 快速笔记功能
  - 收藏题目功能
  - 解题心得分享（可选）

- [x] 8. 搜索与筛选
  - 题目名称搜索
  - 难度筛选
  - 状态筛选（未做/已做/复习中）
  - 标签筛选

- [x] 9. 刷题目标系统
  - 每日刷题目标
  - 每周刷题目标
  - 目标达成提醒
  - 连续刷题天数统计

- [x] 10. 数据导出
  - 导出刷题记录为 JSON/CSV
  - 导出进度到本地存储
  - 备份与恢复功能

## 验收标准

- [x] LeetCode 用户名可成功绑定
- [x] 解题记录可同步到本地
- [x] 推荐算法输出合理结果
- [x] 所有图表数据准确
- [ ] 搜索响应时间 < 100ms

## 备注

- LeetCode API 限制：避免过于频繁的请求
- 考虑添加离线模式支持
- 考虑添加浏览器扩展支持
