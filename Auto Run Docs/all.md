### 1.1 字体硬性配置
- 全站正文/代码/题目/导航/标签/按钮：统一 `Ubuntu Mono`
- 字体兜底：`font-family: 'Ubuntu Mono', Consolas, Monaco, monospace;`
- 像素字体限制：**仅h1顶级大标题可用**（首页主标题、页面顶级标题），禁止用于正文、题目、小字、导航、标签。

### 1.2 固定排版标尺（全局强制统一）
| 元素层级 | 字体 | 字号 | 字重 | 行高 |
| ---- | ---- | ---- | ---- | ---- |
| h1一级大标题 | 像素字体 | 1.8rem | 700 | 1.2 |
| h2二级标题 | Ubuntu Mono | 1.2rem | 600 | 1.4 |
| 正文/题目/代码 | Ubuntu Mono | 1rem | 400 | 1.6 |
| 标签/辅助小字 | Ubuntu Mono | 0.85rem | 400 | 1.5 |

### 1.3 细节落地
1. 删除所有0.35rem/0.38rem极小像素小字。
2. 代码块强制绑定Ubuntu Mono，适配语法高亮。
3. 全平台兼容Mac/Windows/Linux，无乱码、无字体回退失效。

### 验收标准
- 全站正文无像素小字，仅大标题保留像素风；字体清晰专业，终端质感拉满。

## Task2 顶部导航栏彻底重构（解决难看、老旧问题）
### 2.1 布局结构

- 移动端＜768px自动折叠为汉堡菜单，保留全部功能。

### 2.2 样式升级（保留像素风）
- 背景：毛玻璃 `backdrop-filter: blur(12px)`，底色 `rgba(15,23,42,0.8)`。
- 滚动触发：底色加深为`rgba(15,23,42,0.95)`，底部保留1px像素霓虹分割线。
- 删除原有杂乱色块、丑边框、冗余装饰。

### 2.3 导航菜单修改（核心需求）
- 原【文章】永久改名：【分享】，同步修改路由、页面title、跳转链接。
- 最终导航：首页 / 刷题 / 分享 / 关于。
- 右侧固定：语言切换、主题切换（像素小按钮，hover高亮）。

### 2.4 交互规范
- 导航项hover：上浮2px+文字霓虹高亮+像素边框点亮，过渡0.2s。
- 当前选中页：底部3px霓虹像素标识条，文字加粗。
- 移动端汉堡菜单：内置全部导航+主题/语言切换，展开动画顺滑。

### 验收标准

   - 移动端：双列网格，间距16px

   - 选中章节左侧3px霓虹标识，文字加粗；移动端改为横向可滑动标签栏。

### 3.2 章节详情页 chapter-*.html
