一、单调栈

§1.1 基础

单调栈原理讲解

单调栈 (Monotonic Stack)：栈中的元素按照递增或递减的顺序排列。

  • 单调递增栈：栈中元素从栈底到栈顶递增，用于查找最近的小于自己的元素
  • 单调递减栈：栈中元素从栈底到栈顶递减，用于查找最近的大于自己的元素
  • 核心思想：维护一个有序的栈，当遇到不满足顺序的元素时，不断弹出直到满足条件
  • 时间复杂度：O(n)，每个元素最多入栈和出栈一次

Python 模板代码

# 单调递增栈：求最近小于自己的元素下标
def monotonic_increasing_stack(nums):
    n = len(nums)
    left = [-1] * n  # 最近小于自己的元素下标
    stack = []  # 存放下标
    
    for i in range(n):
        while stack and nums[stack[-1]] > nums[i]:
            stack.pop()
        left[i] = stack[-1] if stack else -1
        stack.append(i)
    return left

# 单调递减栈：求最近大于自己的元素下标
def monotonic_decreasing_stack(nums):
    n = len(nums)
    right = [-1] * n  # 最近大于自己的元素下标
    stack = []  # 存放下标
    
    for i in range(n):
        while stack and nums[stack[-1]] < nums[i]:
            stack.pop()
        right[i] = stack[-1] if stack else -1
        stack.append(i)
    return right

Java 模板代码

// 单调递增栈：求最近小于自己的元素下标
public int[] nextSmallerElement(int[] nums) {
    int n = nums.length;
    int[] left = new int[n];
    Arrays.fill(left, -1);
    Deque stack = new ArrayDeque<>();
    
    for (int i = 0; i < n; i++) {
        while (!stack.isEmpty() && nums[stack.peek()] > nums[i]) {
            stack.pop();
        }
        left[i] = stack.isEmpty() ? -1 : stack.peek();
        stack.push(i);
    }
    return left;
}

// 单调递减栈：求最近大于自己的元素下标
public int[] nextGreaterElement(int[] nums) {
    int n = nums.length;
    int[] right = new int[n];
    Arrays.fill(right, -1);
    Deque stack = new ArrayDeque<>();
    
    for (int i = 0; i < n; i++) {
        while (!stack.isEmpty() && nums[stack.peek()] < nums[i]) {
            stack.pop();
        }
        right[i] = stack.isEmpty() ? -1 : stack.peek();
        stack.push(i);
    }
    return right;
}

Go 模板代码

// 单调递增栈：求最近小于自己的元素下标
func nextSmallerElement(nums []int) []int {
    n := len(nums)
    left := make([]int, n)
    for i := range left {
        left[i] = -1
    }
    stack := make([]int, 0)
    
    for i := 0; i < n; i++ {
        for len(stack) > 0 && nums[stack[len(stack)-1]] > nums[i] {
            stack = stack[:len(stack)-1]
        }
        if len(stack) > 0 {
            left[i] = stack[len(stack)-1]
        }
        stack = append(stack, i)
    }
    return left
}

// 单调递减栈：求最近大于自己的元素下标

        }

            right[i] = stack[len(stack)-1]
        }
        stack = append(stack, i)
    }
    return right
}

求最近大于/小于元素的下标

核心应用：单调栈最常见的应用是求每个元素最近的大于或小于自己的元素下标。

  • 求最近小于：使用单调递增栈，从左向右遍历
  • 求最近大于：使用单调递减栈，从左向右遍历
  • 环形数组：将数组拼接两次，模拟环形查找
  • 左边最近：从左向右遍历，维护递增/递减栈
  • 右边最近：从右向左遍历，或从左向右记录结果
